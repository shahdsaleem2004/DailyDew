## Project Overview
DailyDew is a full-stack personalized skincare recommendation platform designed to analyze user inputs and generate customized skincare routines based on structured decision logic and database-driven product data.

Built as a team project, the system integrates a Java Servlet backend with an SQL database to deliver dynamic content, rule-based skin classification, and personalized product recommendations.

## Academic Context

DailyDew was developed as a team-based project for the Web Application Programming course.  

The project focused on applying full-stack development principles, backend logic implementation, and relational database integration in a real-world web application environment.

## Problem Statement

Many users struggle to identify their skin type and select suitable skincare products. Incorrect product choices often lead to irritation, acne flare-ups, or ineffective routines.  

DailyDew addresses this challenge by automating skin analysis and generating intelligent recommendations through structured logic and stored product knowledge.
